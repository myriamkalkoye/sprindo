# Sprindo 

Outil de gestion de projets Scrum développé en Java dans le cadre du cours
INF2050 à l'UQAM.

## Description

Application de gestion agile permettant de gérer des projets, des sprints,
des tâches et des utilisateurs selon la méthodologie Scrum.

## Fonctionnalités

- Création et gestion de projets
- Gestion des sprints et du backlog
- Assignation de tâches aux utilisateurs
- Calcul de statistiques sur les sprints
- Suite de tests JUnit complète

## Technologies

- Java
- JUnit (tests unitaires)
- GitLab CI/CD
- Architecture MVC
